# Type Definitions

This directory contains TypeScript type definitions.

## Types

- **api.types.ts**: API request/response types
- **project.types.ts**: Project-related types
- **repository.types.ts**: Repository-related types
- **common.types.ts**: Common types used across the application

## Guidelines

- Define interfaces for all API responses
- Use type aliases for unions and primitives
- Export all types as named exports
- Keep types DRY (Don't Repeat Yourself)
